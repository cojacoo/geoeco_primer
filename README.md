# Cherry Blossom Data

This is an example of simple data analyses based on the example of blossoming of *prunus jamasakura* in Kyoto, Japan. Aono et al. have collected a spectacular series of phenological data enriched with meteorological data (measured and reconstructed).

For the cherry blossom data: https://ourworldindata.org/grapher/date-of-the-peak-cherry-tree-blossom-in-kyoto?tab=table
For meteorological data: https://www.data.jma.go.jp/stats/etrn/view/monthly_s3_en.php?block_no=47759&view=2

The example is part of the Geoecology Primer class 2025.

Instead of downloading, you can run the example online through [![Binder](https://mybinder.org/badge_logo.svg)](https://mybinder.org/v2/gh/cojacoo/geoeco_primer/HEAD?filepath=blossom26.ipynb).
